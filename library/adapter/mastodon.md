## 環境変数

| キー | 概要 |
| ---- | ---- |
| LXBOT_MASTODON_BASE_URL | mastodonのURLをセットします |
| LXBOT_MASTODON_ACCESS_TOKEN | mastodonのアクセストークンをセットします |
